Rules

If a person, place, organization, or object is listed as "creator-use only" then you cannot use it in your story (unless you are the creator)

If a person, place, organization, or object is listed as "use with creator's permission" then you can use it, so long as you ask the creator if you can use it

If you plan on killing off a character in your story, please tell the creator of that character first (even if you have their permission)

No destroying the world, or blowing up the universe (if you have an idea for that, please tell us)

Please only put PG-13 content in your stories
